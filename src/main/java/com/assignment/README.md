# User List App

This Android application demonstrates the use of MVVM architecture, Retrofit, Room, and geolocation functionalities to display a list of users retrieved from a REST API.

## Features  
- Fetch and display users from https://reqres.in/api/users?page=2.
- Display user avatars, names, and emails in a RecyclerView.
- Get and display the user's current geolocation.
- Allow users to upload an image to replace the avatar.
- Store uploaded images locally using Room.

## How to Run
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the project on an Android device or emulator.

## Assumptions
- The application targets devices running Android 6.0 (Marshmallow) or higher.
- Network availability is required for initial user data fetch.
